# K-Means Plus Flow

Sistem memakai K-Means Plus sebagai metode utama. Frontend tidak menyediakan input jumlah maksimal K. Hasil akhir tetap ditampilkan sebagai 3 tingkatan segmen bisnis:

1. Low
2. Medium
3. High

Kombinasi nilai LRFMC dapat berupa `Low-Medium-High`, `High-Low-Low`, `Medium-High-High`, dan variasi lain.
